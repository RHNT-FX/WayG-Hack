

## Introduction

WayG-Hack dirancang agar mudah diadaptasi untuk berbagai kebutuhan mendapatkan jawaban mudah dari WayGround dan automasi exploitasi.

## Usage

Bagian ini menjelaskan cara menggunakan WayG-Hack tanpa proses instalasi yang rumit. Anda hanya perlu menyalin berkas konfigurasi dan menyesuaikan API AI sesuai kebutuhan.

1. Salin file `console.txt` dari repositori ini ke direktori kerja Anda.
2. Buka file `console.txt` dengan editor teks favorit Anda.
3. Gantilah nilai API AI pada bagian yang disediakan dengan API milik Anda sendiri.
4. Simpan perubahan dan gunakan file tersebut sesuai alur kerja atau tool yang Anda miliki.

## Features

WayG-Hack menawarkan banyak fitur yang berfokus pada automasi, efisiensi, dan kemudahan penggunaan:

- **Auto Answer**
  - Skrip ini bisa otomatis memilih jawaban yang benar dengan akurasi 89%.
  - Dapat mendeteksi soal secara otomatis.

- **Exploit Toolkit**
  - Mudah untuk digunakan.
  - Dapat diperluas dengan modul exploit buatan sendiri.

- **Integrasi API**
  - Dapat menggunakan API Gemini atau API DeepSeek.
  - Mendukung request dan response dalam format JSON.

- **Support untuk Multi-Platform**
  - Dapat dijalankan pada Windows, Linux, dan MacOS.
